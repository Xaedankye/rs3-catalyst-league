Foxlayne - RuneScape Catalyst League Tracker
============================================

INSTALLATION INSTRUCTIONS:

1. Drag Foxlayne.app to your Applications folder
2. If you get "Foxlayne is damaged and can't be opened":
   
   QUICK FIX:
   - Open Terminal
   - Run: xattr -d com.apple.quarantine /Applications/Foxlayne.app
   - Try opening Foxlayne again
   
   ALTERNATIVE:
   - Right-click Foxlayne.app → "Open" → Click "Open" in dialog

3. Launch Foxlayne from Applications folder

This is a one-time fix. Future updates should work normally.

For support, visit: https://github.com/Xaedankye/rs3-catalyst-league
